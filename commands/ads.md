# BenAI Ads Suite

14 specialized paid advertising skills covering Google, Meta, LinkedIn, TikTok, YouTube, and Microsoft. Each is standalone — invoke directly:

## Platform Analysis

| Skill | What it does |
|-------|-------------|
| `/ads-google` | Google Ads deep analysis — 74 checks across Search, PMax, Display, YouTube, Demand Gen |
| `/ads-meta` | Meta/Facebook/Instagram Ads — 46 checks covering Pixel/CAPI, creative, structure, audience, Advantage+ |
| `/ads-linkedin` | LinkedIn Ads for B2B — 25 checks including Thought Leader Ads, ABM, lead gen forms |
| `/ads-tiktok` | TikTok Ads — 25 checks with creative-first strategy, safe zones, Smart+, TikTok Shop |
| `/ads-youtube` | YouTube Ads — skippable, non-skippable, bumper, Shorts, Demand Gen analysis |
| `/ads-microsoft` | Microsoft/Bing Ads — 20 checks with Google import validation, Copilot integration |

## Cross-Platform Tools

| Skill | What it does |
|-------|-------------|
| `/ads-audit` | Full multi-platform audit — runs 6 parallel subagents, aggregate health score (0-100) |
| `/ads-budget` | Budget allocation & bidding — 70/20/10 rule, 3x Kill Rule, scaling assessment |
| `/ads-creative` | Creative audit + brand context setup + ad creative generation via infographic-v2 |
| `/ads-landing` | Landing page optimization — message match, page speed, mobile, forms, trust signals |
| `/ads-plan` | Strategic ad planning — 15 industry templates, campaign architecture, phased roadmap |
| `/ads-competitor` | Competitive intelligence — Ad Libraries, keyword gaps, spend estimation, messaging analysis |
| `/infographic-v2` | Generate branded infographics for ad campaigns and performance reports |

## Getting Started

- **Full account audit:** `/ads-audit` (runs all platforms in parallel)
- **Single platform:** `/ads-google`, `/ads-meta`, `/ads-linkedin`, `/ads-tiktok`, `/ads-youtube`, `/ads-microsoft`
- **Creative audit + generation:** `/ads-creative` (also sets up brand context on first run)
- **Budget review:** `/ads-budget`
- **New campaign planning:** `/ads-plan`
- **Competitor research:** `/ads-competitor`
