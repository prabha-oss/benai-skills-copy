# Reference Example: 30-Day Claude Build Challenge

This is a complete reference example showing how each phase would look for a specific course.

---

## Phase 1 Output: Course Blueprint

### Course Overview
- **Working Title:** 30-Day Claude Build Challenge
- **Tagline:** Build 30 real projects with Claude in 30 days — no coding required
- **Format:** Challenge (cohort-based, daily lessons with daily build tasks)
- **Duration:** 30 days
- **Price Range:** $97 - $297

### Ideal Student
- **Who they are:** Non-technical creators, entrepreneurs, solopreneurs (age 25-55)
- **Current level:** Beginner — heard about AI tools but feel overwhelmed
- **Pain points:** Tried ChatGPT but never built anything real; don't know where to start
- **Motivation:** Want to use AI to save time and create tools for their business
- **Previous attempts:** Watched YouTube tutorials, read blog posts, maybe tried ChatGPT

### Transformation Statement
"By the end of this 30-day challenge, you will have built and shipped 30 real-world projects using Claude — from automations to full apps — even if you've never written a line of code."

### Unique Positioning
- **Angle:** Not another 'intro to AI' course. Daily build challenge — ship something real every day. No fluff. No theory lectures. Just building.
- **Competitors:** Generic AI courses, ChatGPT tutorials, prompt engineering courses
- **Gap:** No course makes you BUILD every single day with Claude specifically

## Phase 2 Output: Curriculum Map

### Week 1 (Days 1-7): FOUNDATIONS
Build understanding of Claude and complete your first projects.

**Day 1: "Meet Claude — Your AI Building Partner"**
- Objective: Explain what Claude is and how it differs from other AI tools
- Content: 10-min video overview
- Activity: Create account, send first message, explore the interface
- Resource: "Claude Quick Start Cheat Sheet" (PDF)

**Day 2: "Your First Build — A Personal Dashboard"**
- Objective: Build a simple HTML artifact using Claude
- Content: 8-min screencast walkthrough
- Activity: Build a personalized dashboard page
- Resource: Starter prompt template

**Day 3: "How Claude Thinks — The Mental Model"**
- Objective: Describe how Claude processes instructions and context
- Content: 12-min video + diagram
- Activity: Experiment with 5 prompting approaches, document results
- Resource: "Prompting Patterns" reference card

**Days 4-7:** Continue building foundations...

### Week 2 (Days 8-14): CORE SKILLS
Master prompting, skills, workflows, and project mode.

### Week 3 (Days 15-21): REAL BUILDS
Create automations, tools, and integrations.

### Week 4 (Days 22-28): ADVANCED + SHIPPING
MCP, plugins, publishing, and scaling.

### Days 29-30: SHOWCASE
Final project presentation + reflection.
