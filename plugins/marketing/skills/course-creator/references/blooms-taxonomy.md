# Bloom's Taxonomy — Complete Verb Reference

Use these action verbs when writing learning objectives for each lesson. Match the verb to the cognitive level appropriate for where the lesson falls in the course.

---

## Level 1: Remember
**Definition:** Recall facts and basic concepts

**Verbs:** define, describe, identify, know, label, list, match, name, outline, recall, recognize, reproduce, select, state

**Use in course:** Early lessons, foundational knowledge, vocabulary, key facts

**Example objective:** "Students will be able to **identify** the three types of Claude projects."

---

## Level 2: Understand
**Definition:** Explain ideas or concepts

**Verbs:** classify, compare, contrast, demonstrate, describe, discuss, distinguish, explain, extend, give examples, illustrate, interpret, paraphrase, predict, rewrite, summarize, translate

**Use in course:** Concept lessons, building mental models, connecting ideas

**Example objective:** "Students will be able to **explain** how context windows affect Claude's responses."

---

## Level 3: Apply
**Definition:** Use information in new situations

**Verbs:** apply, build, calculate, change, choose, compute, construct, develop, discover, execute, experiment, illustrate, implement, interpret, manipulate, modify, operate, practice, predict, produce, relate, schedule, show, sketch, solve, use, write

**Use in course:** Hands-on exercises, following tutorials, reproducing techniques

**Example objective:** "Students will be able to **build** a personal dashboard using Claude's artifact feature."

---

## Level 4: Analyze
**Definition:** Draw connections among ideas

**Verbs:** analyze, break down, categorize, classify, compare, contrast, deconstruct, determine, differentiate, discriminate, distinguish, examine, experiment, infer, inspect, investigate, organize, outline, question, relate, separate, test

**Use in course:** Intermediate lessons, debugging, comparing approaches, examining trade-offs

**Example objective:** "Students will be able to **compare** three different prompting strategies and determine which produces the best output for a given task."

---

## Level 5: Evaluate
**Definition:** Justify a stand or decision

**Verbs:** appraise, argue, assess, choose, conclude, criticize, critique, debate, decide, defend, evaluate, grade, judge, justify, measure, monitor, prioritize, rank, rate, recommend, select, support, test, value

**Use in course:** Advanced decision-making, choosing between tools/approaches, quality assessment

**Example objective:** "Students will be able to **evaluate** whether a Claude-built automation meets production-readiness criteria."

---

## Level 6: Create
**Definition:** Produce new or original work

**Verbs:** assemble, build, compose, construct, create, design, develop, devise, elaborate, formulate, generate, hypothesize, integrate, invent, make, organize, originate, plan, produce, propose, rearrange, revise, synthesize, write

**Use in course:** Final projects, capstones, independent builds, portfolio pieces

**Example objective:** "Students will be able to **design and build** a complete automation workflow that solves a real business problem."

---

## Progression Rule

A well-designed course moves students UP the taxonomy:

```
Modules 1-2:  Remember → Understand  (foundations)
Modules 3-4:  Apply → Analyze        (practice + deeper thinking)
Modules 5+:   Evaluate → Create      (independence + mastery)
```

Every lesson's learning objective should use a verb from the appropriate level. If a lesson in Module 1 uses a "Create" verb, question whether the student has the foundation to create at that point.
