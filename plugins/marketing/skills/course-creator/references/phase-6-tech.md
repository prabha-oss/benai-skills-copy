# Phase 6: Platform & Tech Setup Guide

## Purpose
Tell the user exactly what to set up, where, and in what order.

---

## Platform Recommendation

Based on course format, recommend platforms:

| Format | Recommended Platforms | Why |
|---|---|---|
| Self-paced | Teachable, Thinkific, Kajabi | Easy setup, payment built-in |
| Cohort | Circle, Kajabi, Maven | Community + course in one |
| Challenge | Circle, Skool, Discord + Teachable | Community-first |
| Hybrid | Kajabi, Circle | Flexible |

Provide 2-3 options with pros/cons and let user decide.

---

## Tech Setup Checklist

```markdown
# TECH SETUP CHECKLIST

## Platform & Hosting
- [ ] Choose and set up course platform
- [ ] Configure course structure (modules, lessons)
- [ ] Upload all course content
- [ ] Set up drip schedule (if applicable)
- [ ] Test student experience (enroll as test student)

## Payments
- [ ] Connect payment processor (Stripe/PayPal)
- [ ] Set up pricing (one-time, payment plan, tiers)
- [ ] Create coupon codes (launch discount, early bird)
- [ ] Test checkout flow end-to-end

## Email
- [ ] Set up email marketing tool (ConvertKit, Mailchimp, etc.)
- [ ] Create welcome/onboarding email sequence
- [ ] Create launch email sequence
- [ ] Set up automated triggers (purchase → welcome emails)
- [ ] Test all email sequences

## Community
- [ ] Set up community space (Circle, Discord, Skool, etc.)
- [ ] Create channels/sections (general, introductions, daily-challenges, Q&A, wins)
- [ ] Write community guidelines
- [ ] Create welcome post / pinned messages
- [ ] Test member onboarding flow

## Sales Page
- [ ] Build sales page (on platform or separate landing page)
- [ ] Add payment/CTA buttons
- [ ] Mobile test
- [ ] Set up analytics (Google Analytics, Hotjar)

## Content Delivery
- [ ] Upload all videos / course materials
- [ ] Set up drip schedule (if not all-at-once)
- [ ] Configure completion tracking
- [ ] Set up certificates (if applicable)
- [ ] Test lesson progression

## Legal
- [ ] Terms of service
- [ ] Privacy policy
- [ ] Refund policy
```

---

## Content Delivery Schedule

Based on course format:
- **Self-paced:** All content available immediately (or dripped weekly for engagement)
- **Cohort:** Drip one module per week, synced with live sessions
- **Challenge:** Drip one lesson per day at a consistent time
- **Hybrid:** Core content available, bonus content dripped
