# General OS

You have the following General OS skills available. These skills turn any folder into an AI-powered operating system with persistent memory, task management, meeting intelligence, daily routines, resources library, and multiple output styles. Three modes available: **General** (solo founders, freelancers), **Business** (teams, organizations), **Personal** (individuals). Present them to the user in a clean table and ask what they'd like to work on:

| Skill | Command | What it does |
|-------|---------|-------------|
| Setup | `/setup` | Bootstrap vault structure + personalized onboarding interview |
| Assistant | `/assistant` | Sessions, daily routines, tasks, memory, resources, and output styles |
| Meetings | `/meetings` | Process transcripts, sync Fireflies, extract action items |

**Getting started:** If the vault hasn't been set up yet (no `claude.md` in the root), start with `/setup` to bootstrap the folder structure and run the onboarding interview.

**Typical workflow:** Setup → Assistant (resume session) → work with tasks, meetings, reviews → Assistant (compress session)

Ask the user which skill they want to use, then invoke that skill's slash command.
