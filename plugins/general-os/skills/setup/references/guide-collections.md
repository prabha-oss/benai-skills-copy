# Collections

Your personal curation — books, articles, courses, media, and anything worth remembering.

## How It Works

Collections grow organically. Start with one file per type:

```
Collections/
├── books.md
├── articles.md
├── courses.md
├── podcasts.md
└── movies.md
```

## Collection File Template

```yaml
---
type: collection
tags: [collection]
---

## Currently
- [what you're reading/watching/taking now]

## Completed
| Title | Author/Source | Rating | Key Takeaway |
|-------|-------------|--------|-------------|
| [title] | [author] | ⭐⭐⭐⭐ | [one-liner] |

## Want to
- [items on your list]
```

## Tips
- Don't overthink organization — one file per media type is enough
- Add items as you encounter them
- Link to [[Projects]] or [[Areas]] when something is relevant
- Use ratings to remember what was worth your time
