# Areas

Ongoing areas of responsibility in your life. Unlike projects, areas don't have a finish line — they're things you maintain continuously.

## How It Works

Each area is a single file (not a folder). Keep it simple:

```
Areas/
├── health.md
├── finances.md
├── learning.md
├── career.md
├── relationships.md
└── home.md
```

## Area File Template

```yaml
---
type: area
tags: [area]
last_reviewed: YYYY-MM-DD
---

## Current State
Where things stand right now.

## Standards
What "good" looks like for this area.

## Active Habits
- [ ] [habit related to this area]

## Notes
Observations, plans, things to track.
```

## The Difference: Areas vs Projects

> A **project** has a finish line. An **area** doesn't.
>
> "Run a marathon" is a project. "Stay fit" is an area.
> "Pay off student loans" is a project. "Manage finances" is an area.

## Tips
- Review areas during weekly reviews
- Link areas to related [[Projects]] and [[Goals]]
- Keep area files lean — move detailed plans to Projects/
